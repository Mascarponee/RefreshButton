The main aim of the page is to determine differencies in headers while performing different actions in browser. Also it will be interesting to know when resource comes from cache and when from the server. This page will request several .js files. Each of them will be returned from the server with different headers (cache-related). Headers will be displayed to you, once resource is downloaded. After that you can try different types of browser commands (such as F5, CTRL+R, CTRL+F5, Back/Forward and so on) in order to refresh the page.
